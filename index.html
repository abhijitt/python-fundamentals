Functions


def its_me(name):
    if len(name) > 2 and len(name) < 10 and name != 'Adele':
        # This is what a comment looks like
        # You can even write this instead
        # 2 < len(name) < 10 and name != 'Adele'
        print('Hi')
    elif name == 'Adele':
        """
        This is also a comment. It can span over multiple lines.
        This is equivalent of c++ /* block comment */
        """
        print('Hello')
    elif name[:3] == 'Ada':
        print('Yo')
    else:
        print('Hey')

name = input('Name:\n')
its_me(name)

Common Errors

SyntaxError: invalid syntax 

for x in range (1,10)
    print(x)

Don't forget the colon!


for x in range (1,10):
    print(x)
IndentationError: unexpected indent 

for x in range (1,10):
print(x)


for x in range(0, 5):
	print(x)

Always indent with four spaces. DO NOT USE TABS


for x in range (1,10):
    print(x)
